# Sprint-4
### Para correr un microservicio
1. Navegar al directorio con `cd nombre_del_directorio`
2. Correr `docker-compose up --build` para construir y correr el contenedor
3. Acceder a la aplicación en el puerto especificado (ejemplo: `http://localhost:3000`)

### Para correr todo 
1. Navegar al directorio raíz del proyecto con `cd Sprint-4`
2. Correr `docker-compose -f docker-compose-all.yml up --build` para construir y correr todos los contenedores
3. Acceder a las aplicaciones en los puertos especificados (ejemplo: `http://localhost:3000`)
